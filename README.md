# Smooth Line

A simple demo to draw antialiased lines with GLSL

# Build Instructions

### Visual Studio 2010

 - Open smooth_line.sln

### Mac

 - install SDL2 https://www.libsdl.org/download-2.0.php
 - make -f makefile_mac

### Linux (tested on Ubuntu)

 - install libraries: apt-get install build-essential mesa-common-dev libglu1-mesa-dev libglew-dev libsdl2-dev
 - make -f makefile_linux
